Neon piscando com CSS

Saiba como criar um efeito de neon piscando com CSS.

Pule para o final para o código completo.

HTML

Vamos começar com o html básico:

Nesse efeito, algumas letras irão piscar. Para isso, colocaremos essas letras dentro de <span> tags.

CSS

Vamos aplicar uma fonte light e umas sombras no texto para dar o efeito de neon.

body {
  background-color: #000000;
  color: #FFFFFF;
}

p {
  font-family: Arial, Helvetica, sans-serif;
  font-size: 4.5em;
  font-weight: lighter;
  text-shadow: 
    0px 0px 30px lime,
    0px 0px 20px lime,
    0px 0px 10px lime, 
    0px 0px 5px lime;
}

Passamos 4 valores a cada sombra no text-shadow, sendo o deslocamento no eixo x, deslocamento no eixo y, espalhamento e cor da sombra.

Deixaremos o deslocamento da sombra com 0 em ambos os eixos para que permaneça no meio do texto.

Definimos 4 sombras com espalhamento diferente para obter uma sombra destacada.

Definimos o fundo do <body> preto, para dar destaque ao texto.

Para fazer as letras piscarem, devemos criar uma animação utilizando o @keyframes.

@keyframes animate {
  20%, 36%, 48% {
    text-shadow:
      0px 0px 0px lime;
      0px 0px 0px lime,
      0px 0px 0px lime,
      0px 0px 0px lime;
    color: #818181;
  }
        
  22%, 37%, 50% {
    text-shadow:
      0px 0px 30px lime;
      0px 0px 20px lime,
      0px 0px 10px lime,
      0px 0px 5px lime;
    color: #FFFFFF;
  }
}

A animação alterna o espalhamento das sombras e a cor da letra.

As sombras alternam entre 0 e seu valor inicial, enquanto o texto altera sua cor entre cinza e branco para dar um efeito de luz desligada.

Para animar de fato, utilizaremos a propriedade animation.

p > span:nth-child(1) {
  animation: animate 1s linear infinite;
}
      
p > span:nth-child(2) {
  animation: animate 2s linear infinite;
}
      
p > span:nth-child(3) {
  animation: animate 800ms linear infinite;
}

As três <span> tags devem ser animadas individualmente com tempo de execução diferente, dando assim um efeito de aleatoriedade.
